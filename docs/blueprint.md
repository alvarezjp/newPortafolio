# **App Name**: DevFolioX

## Core Features:

- Hero Section: Display a hero section with the developer's name, title, and a brief introduction.
- Tech Stack Showcase: Showcase the technologies and tools used by the developer, with icons for each technology.
- Project Gallery: Display a gallery of projects with descriptions, images, and links to live demos and GitHub repositories.
- Dynamic Project Cards: Generate project cards dynamically with support for images, descriptions, and links.

## Style Guidelines:

- Primary color: Dark navy blue (#121826) for a professional and modern look, evoking trust.
- Background color: Even darker navy (#0B0F1A) for a sense of depth.
- Accent color: Teal (#45D9D0) for highlights and interactive elements, providing a fresh contrast.
- Body and headline font: 'Inter', a sans-serif, provides a clean, modern, and readable typeface.
- Use simple, line-based icons for technologies and social media links.
- Employ a clean, single-page layout with clear sections for introduction, tech stack, and projects.
- Subtle animations on scroll and hover for interactive elements to improve user experience.
